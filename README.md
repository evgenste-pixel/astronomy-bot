# 🚀 Космический гид — Telegram бот с мини-приложением

## 📋 Что это?
Интерактивный бот-гид по астрономии для детей и взрослых.

## 📦 Как развернуть

### 1. Создайте бота в Telegram
- Напишите @BotFather
- Создайте нового бота командой `/newbot`
- Скопируйте полученный токен

### 2. Загрузите мини-приложение на GitHub Pages
1. Создайте новый репозиторий на GitHub
2. Загрузите в него папку `webapp/` (файлы index.html, style.css, script.js)
3. В настройках репозитория включите **GitHub Pages** (ветка main, папка /root)
4. Скопируйте ссылку (вида `https://username.github.io/repo/`)

### 3. Настройте бота
1. Откройте `config.py`
2. Вставьте токен бота в `BOT_TOKEN`
3. Вставьте ссылку на GitHub Pages в `WEBAPP_URL`

### 4. Запустите бота
```bash
python setup_and_run.py
